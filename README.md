# Tidewise Resilience Studio

Tidewise Resilience Studio is a desktop-style MVP for coastal planners, municipal resilience teams, and nature-based infrastructure analysts. It compares mangroves, oyster reefs, dunes, wetlands, and seawalls across cost, protection, biodiversity, maintenance, carbon, and community benefits.

## MVP Features

- Project setup for coastline length, population exposure, asset value, storm risk, sea-level-rise outlook, and budget.
- Intervention portfolio builder with nature-based and gray infrastructure options.
- Live model outputs for protection, capital cost, annual maintenance, biodiversity uplift, blue carbon, avoided loss, and payback.
- Weighted decision score covering cost, protection, biodiversity, carbon, equity, and maintenance burden.
- Scenario cards, ranked recommendations, and a visual coastal transect mock map.
- Real satellite basemap tiles with GIS-style reef, bleaching, habitat, flood, and critical-asset overlays.
- Generated coastal intelligence visual asset for high-impact planning presentations.
- Local browser persistence for project settings and selected interventions.
- Export to JSON and print-friendly planning brief.

## Run Locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 5173
```

Then visit:

```text
http://localhost:5173
```

## Product Positioning

Tidewise is designed as a first wedge for cities and coastal engineering teams that need an easier way to compare nature-based resilience options before commissioning expensive feasibility studies.
