# Farm Resource Atlas

Farm Resource Atlas is a mobile-first PWA MVP for small farmers who want to plan respectful, low-waste use pathways before resources are lost.

## MVP Features

- Illustrated animal atlas for chicken, cow, and sheep.
- Tappable body-region markers with practical, non-graphic use cases.
- Categories for food, farm inputs, craft/material, feed/byproduct, compost/soil, education, and waste prevention.
- Local saved resource plan with save, remove, clear, copy, print, and JSON export actions.
- Project-local raster animal assets generated for the app.
- Local browser persistence through `localStorage`.
- Installable static PWA with a service worker and manifest.

## Run Locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 5173
```

Then visit:

```text
http://localhost:5173
```

## Product Positioning

The MVP is designed as a visual planning companion for small farms. It avoids graphic process detail and focuses on practical stewardship, sorting, reuse, composting, education, and resource-respect notes.
